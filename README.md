

תוכנית C# לקריאה ועיבוד נתוני תלמידים מקובץ Excel והפקת מכתבי PDF מותאמים אישית באמצעות Word Mail Merge.

---
 מטרת הפרויקט

- לקרוא נתוני תלמידים מקובץ Excel.
- לנקות ולתקן כפילויות ושמות.
- לחשב ציון סופי (60% ציון מעשי + 40% ציון תאורטי).
- להפיק מכתב מותאם אישית לכל תלמיד שעבר את ההכשרה.
- לשמור את המכתבים כ‑PDF.

---
 שלבי הפרויקט

קריאת נתוני Excel וניקוי
- קריאה של קובץ Excel עם עמודות: שם, ציון תאורטי, ציון מעשי, מספר טלפון, דואר אלקטרוני, מחלקה.
- הסרת כפילויות לפי שם.
- תיקון שמות (אות ראשונה גדולה, רווחים תקינים).

 חישוב ציון סופי
- חישוב ציון סופי לפי נוסחה
ציון סופי = 0.6 * ציון מעשי + 0.4 * ציון תאורטי- יצירת הודעה מותאמת אישית (`CustomMessage`):
- אם ציון ≥ 90 → הודעה חיובית על מעבר ההכשרה.
- אם ציון < 90 → הודעה על אי מעבר ההכשרה.

שמירת נתוני Excel מעובדים
- הנתונים המעובדים נשמרים בקובץ חדש (`נתוני_תלמידים_מעובדים.xlsx`) לשימוש בתבנית Word.

הפקת מכתבי PDF באמצעות Word Mail Merge
- פתיחת תבנית Word (`תבנית.docx`) עם **Mail Merge**.
- חיבור ל‑Excel המעובד.
- הפקת PDF לכל תלמיד עם ציון ≥ 70.
- המכתב מותאם לפי השדה `CustomMessage`.
- שמירה בתיקייה `PDFs` עם שם התלמיד
---

הוראות הפעלה

1. יש להכין קובץ Excel עם העמודות הנדרשות.
2. יש להכין תבנית Word עם שדות Merge Field:
   - שם מלא → «שם»
                                                                                                                                                                                  - CustomMessage → «CustomMessage»
   - טלפון → «טלפון»
   - דואר אלקטרוני → «דואר אלקטרוני»
   - מחלקה → «מחלקה»
3. יש לעדכן את הנתיבים בקוד `Program.cs`:
   
                                                                                                                                                       string excelPath = @"C:\Users\Tzivy\Desktop\נתוני_תלמידים.xlsx";
                                                                                                                                         string processedExcelPath = @"C:\Users\Tzivy\Desktop\נתוני_תלמידים_מעובדים.xlsx";
                                                                                                                                                           string wordTemplate = @"C:\Users\Tzivy\Desktop\תבנית.docx";
                                                                                                                                                                  string pdfFolder = @"C:\Users\Tzivy\Desktop\PDFs\";
.
